## Install
Via Composer
``` bash
$ composer require amneale/dns-updater
```

## Testing
Both [phpspec](http://www.phpspec.net) and [behat](http://behat.org) are used to test this library.
``` bash
$ vendor/bin/phpspec run
$ vendor/bin/behat
```
